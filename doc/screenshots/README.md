# Screenshots cần nộp

Sau khi chạy project bằng `npm run dev`, chụp tối thiểu 2 ảnh:

1. Swagger UI tại `http://localhost:5000/api-docs`.
2. GraphQL Sandbox tại `http://localhost:5000/graphql`.

Nên chụp màn hình có thấy rõ endpoint, status code và JSON response để đáp ứng yêu cầu trong `doc/doc.txt`.
